# GlobalSoluction-ErgoMind
Repositório exclusivo para o desenvolvimento de uma API usando Spring boot. 
Essa API foi desenvolvida para ser integrada em uma cadeira ergonômica inteligente equipada com visão computacional, que possui o objetivo de monitorar o usuário, para identificar sinais de fadiga e emitir alertas personalizados que visão contribuir para: 

** Uma vida profisional mais saúdavel 

** Redução de dores e lesões 

** Aumento de produtividade 

** Incentivo a pausas regulares 

** Trabalho mais humano sustentável e inclusivo
